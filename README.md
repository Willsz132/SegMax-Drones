# SegMax Drones

A **SegMax Drones** é uma empresa especializada em soluções com drones para monitoramento, inspeções e segurança, oferecendo tecnologia para otimizar operações em diferentes setores. A landing page foi desenvolvida com foco em apresentar os serviços da empresa de forma clara, destacando seus diferenciais, benefícios e formas de contato.

O projeto possui um design moderno e responsivo, com navegação intuitiva, seções organizadas para apresentação da empresa, serviços, vantagens e chamada para ação (CTA), proporcionando uma boa experiência ao usuário em dispositivos móveis e desktops.

**Principais características:**

* Layout responsivo e moderno.
* Apresentação dos serviços da SegMax Drones.
* Seções com informações sobre benefícios e diferenciais.
* Botões de chamada para ação (CTA) para contato.
* Interface otimizada para uma navegação simples e objetiva.
